## 🚀 Sovereign GCP Deployment
The system is optimized for Google Cloud Run and Cloud SQL.

- **Manual Protocol**: Follow the 10-step guide in [GCP_DEPLOYMENT.md](./GCP_DEPLOYMENT.md).
- **Automated Protocol**: Run `./deploy.sh` for an interactive launch.
- **CI/CD Sharding**: Connect your repository to **Google Cloud Build** using the provided `cloudbuild.yaml`.

## 📦 Core Architecture
- **AI Core**: Google Gemini 3 Pro/Flash Ensemble.
- **Data Shard**: Managed PostgreSQL via Prisma.
- **Execution**: Angel One SmartAPI Production Bridge.
- **Security**: AES-256 GCM Client-side Encryption.
